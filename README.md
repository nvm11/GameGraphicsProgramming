# Graphics Framework Created Utilizing [D3D11](https://learn.microsoft.com/en-us/windows/win32/direct3d11/atoc-dx-graphics-direct3d-11)

## Technologies Used
* C++
* D3D11
* HLSL

## Current Capabilities
* Rasterization with PBR and Blinn-Phong
* Material Creation
* Paralax Mapping
* Box Blur Post-Process
* Cubemap Relfections
* Reading in .obj Files
* Perspective and Orthogonal Cameras
* ImGui Support

## Current Missing Features
* LOD support
* Advanced Reflection Techniques (ex: Reflection Probes)
* Transparency
* Interior Mapping
